# PR 作成スキル

## Overview

Pull Request を作成する。必要に応じて push も行う。

## 手順

1. 既にコミットが push されているか確認し、push されていない場合は push する
2. Pull Request を作成する。Pull Request の作成には gh コマンドを使う

## 注意事項

- あなたは手順にある以外の操作はしてはいけません
  - 何か実装に不具合がある場合もそのまま Pull Request を作成してください
  - 動作確認などは不要です

- Pull Request の本文も記載してください
  - Pull Request の本文を書く際はコミット履歴や変更内容、関連 issue を参考にしてください

- デフォルトブランチに注意してください
  - このプロジェクトではデフォルトブランチが main 以外に設定されている可能性があります
